# Awesome Pearl x n8n Templates

A curated collection of **n8n workflow templates** that integrate with **Pearl Hybrid Intelligence** to combine AI assistance with **professional expert verification**.

## What you’ll find here
- Ready-to-import **n8n workflow JSON** templates
- Patterns for:
  - AI intake + clarification
  - Draft answer generation
  - Expert verification via Pearl MCP Server
  - Stateless webhook-based flows (caller manages conversation history)
- Clear in-workflow documentation via Sticky Notes

## Why Pearl Hybrid Intelligence
Pearl Hybrid Intelligence enables workflows where an AI assistant can draft an answer and then route it to a **qualified professional expert** for verification—ideal for higher-stakes or high-trust user experiences across many domains.
